# Тестирование

В проект добавлены автоматизированные backend-тесты на встроенном модуле Node.js `node:test`.

Запуск:

```bash
npm test
```

Что проверяется:

1. `/api/health` возвращает статус API и счетчики сущностей.
2. Валидация не пропускает некорректное бронирование.
3. Гость может создать бронирование и прочитать его по `accessToken`.
4. Администратор может войти и выполнить CRUD-операции с отелем.
5. Администратор может изменить бронирование и прочитать журнал аудита.

Для тестов создается временная JSON-БД в системной папке, поэтому реальные данные проекта не портятся.
