# Проверка требований курсовой работы

## Требования на оценку 3

Выполнено:

- есть frontend;
- есть отдельный backend;
- реализован REST API;
- есть CRUD для сотрудников;
- есть валидация данных;
- есть структурированное хранение данных;
- есть понятные ошибки;
- есть инструкция запуска;
- система не завершается аварийно при ошибках пользователя.

## Требования на оценку 4

Выполнено:

- есть роли и разграничение доступа;
- есть журнал действий пользователей;
- логика представления отделена от бизнес-логики backend;
- используются понятные имена файлов, функций и сущностей;
- есть backend-тесты;
- есть документация API, архитектуры, запуска и тестирования;
- есть резервное копирование базы;
- есть интеграционная совместимость с основным сайтом Roomly.

## Элементы на оценку 5

Частично выполнено:

- подготовлена интеграционная точка с основным сайтом;
- есть отчет по нагрузке сотрудников;
- есть расширенная модель управления персоналом.

Эти элементы не перегружают проект и могут быть использованы на защите как дополнительный плюс.
