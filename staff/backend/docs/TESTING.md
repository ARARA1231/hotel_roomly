# Тестирование

Тесты запускаются командой:

```bash
npm test
```

Проверяются следующие сценарии:

1. Backend отвечает на служебный запрос `/api/health`.
2. Администратор может войти и получить список сотрудников.
3. Валидация не пропускает некорректные данные сотрудника.
4. Backend не позволяет создать пересекающуюся смену.
5. Пользователь с ролью `staff` видит только свои задачи.

Эти проверки покрывают основные бизнес-процессы панели менеджмента персонала.
